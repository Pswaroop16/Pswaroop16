## 🚀 Featured Projects

### 🎬 Multimodal Content-Based Video Retrieval System

> *Retrieve the most relevant 2–3 second video clips from large databases using image or video queries*

**What it does:** Given an image or a short video clip as a query, the system identifies and returns the most semantically similar clips from a large video corpus — without relying on metadata or text labels.

**Technical approach:**
- Extracted per-frame and temporal features using transformer-based encoders
- Generated cross-modal embeddings that map images and videos into a shared semantic space
- Implemented similarity ranking with optimized nearest-neighbor search
- Built end-to-end indexing pipelines for large-scale video datasets

### 🔍 OWL-ViT Based Open-Vocabulary Vision System

> *Detect and retrieve any object described in plain English — no predefined class list required*

**What it does:** Uses OWL-ViT (Open-World Localization Vision Transformer) to detect objects in images based on natural language descriptions, enabling zero-shot recognition of arbitrary object categories.

**Technical approach:**
- Fine-tuned OWL-ViT for domain-specific open-vocabulary detection
- Built a natural language query interface for image-based semantic search
- Demonstrated zero-shot generalization to unseen object classes
- Integrated vision-language alignment for precise localization
